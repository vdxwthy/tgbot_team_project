# 🎬 Проект: Кино-Менеджер (Telegram Bot)

Интеллектуальный чат-бот для поиска информации о фильмах, генерации кратких пересказов сюжета и автоматической классификации жанров.

## 🌐 Использованные API
В проекте задействованы следующие внешние сервисы:
- **OMDb API (Open Movie Database)**: Используется для получения детальной информации о фильмах (название, год выпуска, рейтинг IMDb, постеры и полные описания сюжета).
- **OpenRouter API**: Обеспечивает доступ к современным языковым моделям (LLM). Используется для перевода поисковых запросов и описаний, а также для генерации интеллектуальных саммари (кратких пересказов).

## 🧠 Базовая модель ИИ
Для задач обработки естественного языка (NLP), таких как перевод и генерация текста, используются модели через OpenRouter. Основной рабочей моделью является:
- **google/gemini-2.0-flash-001**: Высокопроизводительная модель, обеспечивающая быструю и качественную генерацию кратких пересказов сюжета.
- *Резервные модели:* openrouter/healer-alpha, nvidia/nemotron-3-super-120b-a12b:free (для обеспечения отказоустойчивости).

## ⚡ Модель Трансформера и задачи
Для локальной классификации текстов и определения жанров используется специализированная модель архитектуры Transformer:
- **Название модели**: `google-bert/bert-base-uncased` (интегрирована локально в проект).
- **Решаемая задача**: **Классификация текста (Text Classification)**. 
- **Описание**: Модель анализирует текстовое описание фильма (Plot) и на основе семантического анализа определяет наиболее подходящие жанры (например, Фантастика, Комедия, Драма и т.д.). Использование локальной модели позволяет боту работать быстрее и не зависеть от внешних лимитов API при определении жанров.
